## Overview

A 3D singleplayer horror game, based on local folklore of the Overijssel region in the Netherlands. 

Escape the glanner nun while traversing a haunted monastery, while in search for your missing brother.

## Features

### Gameplay (Engineers)

- **Inventory and item system:** Player can pick up and use items to solve puzzles / interact with objects.
- **Enemy AI:** A moving statue that chases the player and interacts with light.
- **Light/Lantern mechanics:** Player uses lantern that emits light. The enemy avoids the light and only attacks the player when he is in complete darkness.
